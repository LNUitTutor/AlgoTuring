# Емулятор машини Тюрінга
Програма імітує роботу однострічкової одноголівкової машини.

Програму - послідовність команд вигляду $q_i s_i$ -> $s_j q_j Move$ - можна задати "вручну" за допомогою `setCommand(q,s,cmd)` або завантажити з файлу.

Порожній символ позначено тильдою.

Приклади використання є в головній програмі.
