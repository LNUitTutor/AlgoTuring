# Емулятор машини Тюрінга
Програма імітує роботу однострічкової одноголівкової машини.

Програму - послідовність команд вигляду $q_i s_i \rightarrow s_j q_j Move$ - можна задати "вручну" за допомогою `setCommand(q,s,cmd)` або завантажити з файлу.

Порожній символ позначено тильдою. Розташування голівки відображено інверсними кольорами виведення на консоль (для цього використано SetConsoleTextAttribute() - працює тільки у Windows).

Приклади використання є в головній програмі.
